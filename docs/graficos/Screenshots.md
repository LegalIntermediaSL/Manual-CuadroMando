### Cómo actualizar estas capturas

Para administradores del sistema:

1. **Captura**: Abrir el informe en Power BI Desktop o Power BI Service.
2. **Formato**: Usar la herramienta de recortes (`Win+Shift+S` o `Cmd+Shift+4`).
3. **Nombre**: Guardar los archivos de imagen en la carpeta `docs/graficos/` con nombres descriptivos.
4. **Referencia**: Actualizar este archivo Markdown vinculando los nuevos archivos.
