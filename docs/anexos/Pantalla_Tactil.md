# Anexo IV: Pantalla Interactiva Samsung Flip 65"

El CMI-DAC ha sido diseñado para ser no solo una herramienta de análisis individual, sino un **catalizador de reuniones estratégicas**. Para ello, la Sala de Juntas Principal está equipada con una pantalla táctil de gran formato.

---

## 1. Equipamiento: Samsung Flip 65"

Se utiliza un monitor interactivo **Samsung Flip (WMR Series)** de 65 pulgadas con resolución 4K UHD. Este dispositivo transforma la experiencia de presentar datos pasivos en una sesión de trabajo colaborativa.

### Características Clave para el CMI

* **Tecnología Táctil**: Permite tocar directamente los gráficos de Power BI para filtrar y profundizar en los datos, tal como se haría en una tablet gigante.
* **Resolución 4K**: Garantiza que los textos pequeños y las cifras detalladas del cuadro de mando se lean con nitidez desde cualquier punto de la mesa de reuniones.
* **Modo Lápiz**: Permite hacer anotaciones manuscritas sobre el propio Dashboard congelado para resaltar puntos clave durante una discusión.

---

## 2. Ventajas en Reuniones Directivas

El uso de este hardware específico aporta valor tangible a las sesiones de seguimiento:

### 2.1. Dinamismo "Minority Report"

En lugar de un presentador estático con un puntero láser y un PowerPoint "muerto", el ponente interactúa físicamente con los datos.

* *Ejemplo: El CEO puede tocar la barra de "Marbella" y todo el informe se recalcula instantáneamente para mostrar solo datos de esa oficina.*

### 2.2. Efecto "War Room" (Sala de Situación)

La gran superficie visual permite tener múltiples métricas a la vista simultáneamente, facilitando la detección de correlaciones (ej. bajada de visitas vs. aumento de precio medio).

### 2.3. Eliminación de Barreras

Al ser un dispositivo táctil intuitivo, cualquier asistente a la reunión puede levantarse e interactuar con el cuadro de mando para responder a una pregunta, democratizando el acceso al dato.

---

## 3. Guía de Conexión y Uso

Para sacar el máximo partido en una presentación de Power BI:

### Opción A: PC Dedicado (Recomendada)

1. Conecte el Mini-PC de la sala al puerto **HDMI 1** y el cable **USB-Touch**.
2. Esto permite que los toques en la pantalla controlen el ratón de Windows 11.
3. Abra el Dashboard en modo "Pantalla Completa" en Power BI.

### Opción B: Proyección Inalámbrica (Smart View)

1. Desde un portátil o tablet, pulse `Windows + K` o use "Smart View".
2. Seleccione la pantalla "Samsung Flip".
3. *Nota: En este modo la respuesta táctil puede tener un ligero retardo (lag), por lo que se recomienda para visualización pasiva.*

---

## 4. Mantenimiento y Cuidado

* **Limpieza**: Use solo paños de microfibra. No aplique líquidos directamente sobre el panel.
* **Stylus**: Utilice el lápiz magnético suministrado (o la punta del dedo). No use bolígrafos reales ni objetos punzantes.
